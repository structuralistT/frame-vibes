# Frame-vibes
Скромный интерактивный веб-инструмент для моделирования и расчета плоских строительных рам, балок и ферм методом конечных элементов (МКЭ).

**🔗 Попробовать в браузере:** [frame-vibes-tobehappy.streamlit.app](https://frame-vibes-tobehappy.streamlit.app/)

Проект написан исключительно ради интереса, вайбкодинга и практики. Основная цель — быстрая и удобная автоматизация проверки домашних заданий и расчетно-графических работ по строительной механике и сопромату прямо в браузере.

## Основные возможности

Несмотря на развлекательный статус проекта, была попытка сохранить логику, защищающую от типичных ошибок ввода:

* **Интерактивный графический редактор:** Создание узлов, стержней и врезных шарниров. Топология графа полностью построена на уникальных `UUID`, что исключает путаницу при пересчете элементов.
* **Автоматический МКЭ-расчет:** В один клик вычисляются опорные реакции и строятся эпюры изгибающих моментов ($M$), поперечных ($Q$) и продольных ($N$) сил (благодаря интеграции с движком `anaStruct`).
* **Модуль метода сил:** Автоматически находит продольные затяжки в системе, осуществляет их мысленный разрез для формирования основной системы метода сил, составляет систему канонических уравнений и выводит подробный пошаговый ход решения в красивых формулах $\LaTeX$.
* **Перемещения по Мору / Верещагину:** Численное перемножение эпюр для определения деформаций в заданных узлах.


##  Стек технологий

* **Python 3.11+**
* **Streamlit** — легкий веб-интерфейс, отзывчивый к изменениям состояния.
* **anaStruct** — двумерный расчетный FEA-движок.
* **Matplotlib & NumPy** — для высокоточного рендеринга инженерных эпюр и матричных операций.
* **Pandas** — для структурированного вывода таблиц реакций и деформаций.

## 🚀 Как запустить проект локально

1. Скопируйте репозиторий с проектом:
   ```bash
   git clone [https://github.com/structuralistT/frame-vibes.git](https://github.com/structuralistT/frame-vibes.git)
   cd frame-vibes
   
 Установите необходимые библиотеки (рекомендуется использовать виртуальное окружение venv):
    Bash

    pip install -r requirements.txt

Запустите Streamlit-приложение гарантированным способом через модуль интерпретатора:
Bash

    python -m streamlit run app.py

Структура репозитория

    app.py — основная точка входа, конфигурация интерфейса, вкладок и боковой панели Streamlit.

    engine.py — вычислительное ядро. Сборка системы, триггеры расчетов, генерация матриц и LATE​X-отчетов метода сил.

    state_manager.py — CRUD-логика для управления узлами и элементами внутри st.session_state.

    models.py — датаклассы и строгие типы данных (TypedDict) для описания геометрии.

    plotting.py — кастомизация стилей Matplotlib, отрисовка затяжек, шарниров и эпюр.
